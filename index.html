数当てゲーム

数当てゲームは、1から10までの整数の中からランダムに選ばれた正解の数字をプレイヤーが予想し、当てるゲームです。毎回解答するたびに「答えはもっと大きいよ」「答えはもっと小さいよ」といったヒントが表示され、それをもとに答えを絞っていきます。また、正解までの回数もカウントされます。正解すると「正解です！」というメッセージが表示されます。4回目以降になるとゲーム終了の案内も表示される仕組みになっています。

数当てゲームのページ: kazuate.html

各授業回の学習事項

第1回 WWW と HTML

今回の講義ではHTMLの記述について学びました。とても面白かったです。自分の好きなるサイトなどを作れるようになりたいと思いました。もっと勉強してクオリティの高いもの作りたいです。まだどのタグを使うといいかわからない所もあるので早く覚えたいです。

演習の成果: 演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

今回の講義ではcssついて学びました。細かいところまでこだわってる来ることができるのでやり甲斐があるなと思いました。高さや幅を決めるのが苦労しました。ちょうどいい大きさのものが作れて満足しています。また、影をつけたりグラデーションをつけたりなどもっとクオリティの高いものにできるように頑張りたいと思います。

演習の成果: 演習2-3: ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

今回の講義では乱数で決めた答えに対し、ユーザの予想値との比較を行う数当てゲームの判定処理をJavaScriptで実装しました。回答回数に応じた条件分岐や、正解・不正解の判定処理を行い、結果をコンソールに出力することでプログラムの流れを確認できました。実際に条件によって表示内容が変化するのが面白く、JavaScriptの基本的な考え方を学ぶ良い機会になりました。

演習の成果: 演習3-6 オブジェクトの参照

第4回 DOM と JavaScript

今回の講義では、JavaScriptを使って数当てゲームを作成したりprojectを編集したりして、JavaScriptやDOM操作を学びました。数当てゲームではユーザーの入力を取得し、コンソールに反映させたり、正解かどうかを判定してヒントや結果を画面に表示することが出来ました。HTML要素をJavaScriptで動的に操作する方法を体験できて良かったです。実際に自分でコードを書いてみて、動くものを作る楽しさと達成感がありました。

演習の成果: 演習4-10 足し算ページ

第5回 イベント処理

今回の講義では、DOMの高度な機能として要素の作成・追加・削除・属性の変更などを学びました。createElement や insertAdjacentElement、setAttribute、remove などを使って、JavaScriptでページ内容を動的に操作する方法を理解することができました。初めは少し難しく感じましたが、実際に手を動かして試すことで、DOM操作のしくみがよく分かり、とても勉強になりました。

演習の成果: 演習5-7 DOMプログラミング(2)
