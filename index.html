「Webコンテンツ作成」の成果

本テーマについて

HTMLやCSS、JavaScriptの基本的な構文と機能を理解し、実際にWebページを構築・編集できる力を身につけることです。Webコンテンツ作成を通して、情報を的確に伝えるレイアウト設計や視覚的デザインの基礎、ユーザーインターフェースの考え方を学ぶことを目標とします。

プロジェクト課題について

このプロジェクト課題では、番組表を検索できるWebアプリケーションを作成しました。ユーザーがチャンネルとジャンルを選択して検索ボタンを押すと、該当する番組が表示されます。JavaScriptでfetch関数を使ってJSONデータにアクセスし、条件に合う番組のみを抽出・表示します。検索結果の見やすさにも配慮し、番組名や放送時間、出演者などの情報を整えて表示しています。前の検索結果を消去してから新しい結果を表示するよう工夫し、繰り返し検索しても見やすくなるようにしました。HTML・CSS・JavaScriptの基礎を応用しながら、Ajax通信とDOM操作の仕組みを理解し、実践的なスキルを身につけることができました。

プロジェクト課題で作成したページ: project.html

数当てゲーム

この数当てゲームは、1から10までの整数の中からコンピュータがランダムに選んだ正解の数字をプレイヤーが予想し、当てるシンプルなゲームです。プレイヤーが予想を入力して「判定」ボタンを押すと、現在の挑戦回数が表示され、入力した数字と正解を比較して「まちがい。答えはもっと大きいよ」「まちがい。答えはもっと小さいよ」といったヒントが出されます。それを参考にしながら、プレイヤーは答えを絞っていきます。正解すると「正解です！」と表示され、それ以上の入力はできなくなります。4回目の挑戦以降になると、正解に関係なく「答えは〇〇でした。ゲームは終了です。」と表示され、ゲームが強制終了します。挑戦回数をカウントしながらヒントを出し、プレイヤーが論理的に答えを導き出すことができる構成になっています。

数当てゲームのページ: kazuate.html

各授業回の学習事項

第1回 WWW と HTML

今回の講義ではHTMLの記述について学びました。とても面白かったです。自分の好きなるサイトなどを作れるようになりたいと思いました。もっと勉強してクオリティの高いもの作りたいです。まだどのタグを使うといいかわからない所もあるので早く覚えたいです。

演習の成果: 演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

今回の講義ではcssついて学びました。細かいところまでこだわってる来ることができるのでやり甲斐があるなと思いました。高さや幅を決めるのが苦労しました。ちょうどいい大きさのものが作れて満足しています。また、影をつけたりグラデーションをつけたりなどもっとクオリティの高いものにできるように頑張りたいと思います。

演習の成果: 演習2-3: ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

今回の講義では乱数で決めた答えに対し、ユーザの予想値との比較を行う数当てゲームの判定処理をJavaScriptで実装しました。回答回数に応じた条件分岐や、正解・不正解の判定処理を行い、結果をコンソールに出力することでプログラムの流れを確認できました。実際に条件によって表示内容が変化するのが面白く、JavaScriptの基本的な考え方を学ぶ良い機会になりました。

演習の成果: 演習3-6 オブジェクトの参照

第4回 DOM と JavaScript

今回の講義では、JavaScriptを使って数当てゲームを作成したりprojectを編集したりして、JavaScriptやDOM操作を学びました。数当てゲームではユーザーの入力を取得し、コンソールに反映させたり、正解かどうかを判定してヒントや結果を画面に表示することが出来ました。HTML要素をJavaScriptで動的に操作する方法を体験できて良かったです。実際に自分でコードを書いてみて、動くものを作る楽しさと達成感がありました。

演習の成果: 演習4-10 足し算ページ

第5回 イベント処理

今回の講義では、DOMの高度な機能として要素の作成・追加・削除・属性の変更などを学びました。createElement や insertAdjacentElement、setAttribute、remove などを使って、JavaScriptでページ内容を動的に操作する方法を理解することができました。初めは少し難しく感じましたが、実際に手を動かして試すことで、DOM操作のしくみがよく分かり、とても勉強になりました。

演習の成果: 演習5-7 DOMプログラミング(2)

工夫点，苦心点，強調点，感想

この授業ではHTML、CSS、JavaScriptの基本から応用までを学び、実際に手を動かしながらWebページを制作することで実践的なスキルを身につけました。特に印象に残っているのは、JavaScriptでfetchを使って外部のJSONデータを取得し、条件に応じて表示を切り替える機能を作成したことです。プロジェクト課題では、提供されたテレビ番組のJSONデータをもとに、ユーザーがチャンネルとジャンルを選択して検索ボタンを押すと、条件に合った番組だけが画面に表示されるWebアプリを実装しました。前回の検索結果が残らないように毎回DOMから削除する処理を入れ、使いやすさにも配慮しました。また、数当てゲームの作成では、条件分岐・変数・イベント処理などJavaScriptの基本構文を使って、回数カウントやヒント表示のロジックを組み立てる経験ができました。CSSでは、文字の大きさや色、余白や配置の調整方法などを学び、自分の作ったページの見た目を整えることができるようになりました。演習課題を通してHTMLの基本構造やタグの意味、DOM操作の考え方なども徐々に理解が深まりました。最初は難しかった構文も、繰り返し手を動かすことで自然と身についていきました。今回の学びを通して、Web制作はただ情報を並べるだけでなく、ユーザー目線での工夫や設計が重要だと強く感じました。特に、情報の構造化と視認性の向上、そして操作性を意識した作り方の重要性を体験的に学べたことが大きな収穫でした。さらに、デベロッパーツールを使ってHTMLやCSSのどの部分が画面にどのように反映されているかを確認する方法も覚えることができ、ブラウザ上での見え方の違いや不具合の原因を探すための視点も身につきました。また、ボタンをクリックしたり入力したりすると、その動きに応じて画面の内容が変わるようなしくみも作ってみました。自分の操作で画面が変わるのはとても楽しくて、プログラムの面白さを実感しました。演習を通じて、目で見える結果がすぐに分かることがモチベーションとなり、繰り返しコードを書く習慣が身につきました。課題ごとに新しい技術を試すたびに理解が深まり、少しずつですが自信もついてきました。今後はアニメーションやフォーム処理など、より実用的な技術にも挑戦し、自分のアイデアを自由に表現できるようになりたいです。さらに、他の人が見ても使いやすいページを目指して、可読性の高いコードを書くことや、ユーザーが直感的に操作できるUI設計にも意識を向けていきたいと考えています。
